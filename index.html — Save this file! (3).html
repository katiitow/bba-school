🗄️ Connect Your Free Firebase Database

Firebase is a free database by Google. All your students, grades, and attendance will be saved permanently and synced across all devices. Follow these steps:

📋 Step-by-Step Setup (Free)

1

Go to console.firebase.google.com → Sign in with your Google account.

2

Click Add project → Name it bba-school → Click Create project.

3

Click Realtime Database (left menu) → Create Database → Choose your region → Start in Test mode → Enable.

4

Click the ⚙️ gear icon → Project settings → Scroll down → Click Web app icon (</>) → Register app → Copy the firebaseConfig object.
